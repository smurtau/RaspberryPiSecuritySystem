Configuration files for Raspberry Pi running archlinux.
Run the script to install dependencies, then place the config files here in the following locations:

/etc/ssh/sshd_config
/etc/snort/snort.conf
/etc/sysctl.d/sysctl.conf
/etc/dnsmasq.conf
/etc/firewall.basic
/etc/firewall.flows
